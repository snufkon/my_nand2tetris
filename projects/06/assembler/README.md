# assembler

Assembler program written by Clojure.

ref: http://www.nand2tetris.org

## Usage

**Create jar file**

`$ lein uberjar`

**Run**

`$ java -jar ./target/assembler-0.1.0-SNAPSHOT-standalone.jar filepath`
